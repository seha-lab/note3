엘리베이터 도착 시간 예측 알고리즘

이 웹페이지는 엘리베이터의 현재 위치, 이동 방향, 외부 호출 상태, 내부 목적층 정보를 바탕으로 사용자가 있는 층까지 엘리베이터가 도착하는 예상 시간을 계산합니다

1. 문제 인식

엘레베이터를 기다리는 것은 지루하다 특히 엘레베이터가 어디를 들렀다가 오는 지 얼마 후에 도착 할 지 알수없어 불편하다

2. 알고리즘의 기본 개념

본 시스템은 사용자가 엘리베이터 버튼을 누르는 순간, 현재 엘리베이터의 정확한 위치와 이동 방향, 그리고 이미 접수되어 있는 다른 층의 요청 정보, 엘레베이터 내부의 호출 데이터 분석하여 사용자가 기다려야 하는 시간을 예측하는 알고리즘입니다.

구체적 알고리즘

엘레베이터가 상향 이동 중일 때의 계산

엘레베이터가 들를 층의 순서를 엘레베이터이동순서 라고 정의한다 현재층보다 높은 상향호출층집합을 낮은 층부터 정렬한것을 엘레베이터이동순서에 추가한다 하향호출층집합을 높은 층 부터 정렬한 것을 엘레베이터이동순서에 추가한다 현재층보다 낮은 상향호출층집합을 낮은 층부터 정렬한 것을 엘레베이터이동순서에 추가한다

엘레베이터가 하향 이동 중일 때의 계산

하향 이동 중일 때는 위의 구조를 반대로 적용한다. 엘레베이터가 들를 층의 순서를 엘레베이터이동순서 라고 정의한다 현재층보다 낮은 하향호출층집합을 높은 층부터 정렬한것을 엘레베이터이동순서에 추가한다 상향호출층집합을 낮은 층 부터 정렬한 것을 엘레베이터이동순서에 추가한다 현재층보다 높은 하향호출층집합을 높은 층부터 정렬한 것을 엘레베이터이동순서에 추가한다

정지 상태일 때

엘리베이터가 정지 상태라면 새로 들어온 호출 위치를 기준으로 방향을 결정한다. 요청한 층이 현재층보다 높으면 상향, 낮으면 하향으로 방향을 정하고, 같은 층이라면 즉시 문을 열어 사용자가 탑승할 수 있도록 한다.

3. 도착 시간 예상 과정

엘리베이터 이동 순서를 기반으로 실제 이동 시간을 계산합니다.

현재 엘리베이터의 층과 엘리베이터 이동 순서의 첫 항(가야할 층) 사이의 거리를 구한다.

거리를 이용해 이동 시간을 계산한 뒤, 각 층에서의 대기 시간을 더한다.

다음 층부터 마지막 층까지 동일한 과정을 반복하여 각 구간의 시간을 합산하고, 총 도착 예상 시간을 구한다.

4. 층 사이 거리별 시간 계산 공식

등속운동이 있는 경우 (층간거리가 충분히 긴 경우)

가속 구간:

엘리베이터가 출발하여 최대속도 v에 도달할 때까지 가속도 a로 이동

가속거리 = v² / (2a)
가속시간 = v / a
등속 구간:

가속과 감속을 제외한 나머지 구간을 등속으로 이동

등속거리 = 전체거리 - (가속거리 + 감속거리)
등속시간 = 등속거리 / v
감속 구간:

목적 층에 가까워지면 감속도 b로 정지

감속거리 = v² / (2b)
감속시간 = v / b
총 이동 시간:
총시간 = 가속시간 + 등속시간 + 감속시간 + 각 층 정차 대기시간

등속운동이 없는 경우 (층간거리가 짧아 최대속도에 도달하지 못하는 경우)

가속과 감속 거리 계산:

총거리 s에 맞춰 가속거리 s₁과 감속거리 s₂를 거리 비율로 조정

s₁ = (b / (a + b)) × s
s₂ = (a / (a + b)) × s
최고점 속도 계산:
v_max = √(2 × a × s₁) = √(2 × b × s₂)
각 구간 시간 계산:
t_acc = v_max / a
t_dec = v_max / b
총 이동시간 = t_acc + t_dec + 각 층 정차 대기시간
