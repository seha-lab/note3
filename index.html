2-2. 도착 시간 예측 과정

엘리베이터 이동 순서를 기반으로 실제 이동 시간을 계산합니다.

현재 엘리베이터의 층과 엘리베이터 이동 순서의 첫 항(가야할 층) 사이의 거리를 구한다.

거리를 이용해 이동 시간을 계산한 뒤, 각 층에서의 대기 시간을 더한다.

다음 층부터 마지막 층까지 동일한 과정을 반복하여 각 구간의 시간을 합산하고, 총 도착 예상 시간을 구한다.

2-3. 층 사이 거리별 시간 계산 공식

등속운동 있는 경우:

가속 → 등속 → 감속

가속거리 = v² / (2a)
등속시간 = 등속거리 / v
감속시간 = v / b
총시간 = 가속시간 + 등속시간 + 감속시간 + 정차대기시간
등속운동 없는 경우:

가속과 감속만 존재, 최대 속도에 도달하지 못함

s₁ = (b / (a + b)) × s
s₂ = (a / (a + b)) × s
v_max = √(2 × a × s₁) = √(2 × b × s₂)
총 이동시간 = t_acc + t_dec + 정차대기시간
